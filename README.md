## Quick start

1. Install dependencies: `npm ci`
2. Start dev server: `npm run dev`
3. Lint/type-check/tests:
   - `npm run lint`
   - `npm run type-check`
   - `npm run test`
4. Build: `npm run build`

More details in the documentation.
